Como funciona a minha privacidade?

Seus dados são guardados num banco de dados que é próprio da marca Guxtavo. o banco de dados é bem seguro, quanto a isso pode ficar tranquilo!

Quais dados meus são guardados?

Nós guardamos seu nome de usuário e senha, guardamos o ID do seu Instagram apenas para poder fazer ações em sua conta automaticamente.
